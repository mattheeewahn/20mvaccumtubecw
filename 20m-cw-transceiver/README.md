# 20m Band CW Transceiver (14MHz, 50W Output)
## 진공관 + IC 하이브리드 설계

### 설계 목표
- **주파수**: 14.000 ~ 14.070 MHz (CW 구간)
- **출력**: 50W PEP
- **모드**: CW (전신)
- **구성**: 진공관(PA) + IC(VFO, 수신부)
- **목표 교신**: 일본, 중국, 호주, 유럽, 미국 (DX)

## 각 단 상세 회로 설계

### 1. VFO (가변 주파수 발진기)

**방식**: NE602 내부 발진기 활용 또는 별도 Colpitts VFO

**부품**:
- IC: NE602AN (SA612AN) — 더블밸런스드 믹서 겸 발진기
- 바리캡: BB109 또는 1SV149
- VFO 코일: T37-6 토로이드 코어, 0.3mm 에나멜선 20회
- C: 폴리스티렌 콘덴서 (온도 안정성)
  - 100pF, 47pF, 22pF (NPO/C0G)
- 트리머: 5~60pF 세라믹 트리머
- 주파수 범위: 5.0 ~ 5.07 MHz (IF 9MHz 사용 시)

```
VFO 회로 (Colpitts 방식):
                    +9V
                     │
                     R1 (100Ω)
                     │
              ┌──────┴──────┐
              │    NE602     │
              │              │
Pin6 ────C1───┤ OSC IN      │
(100pF)       │              ├──── Pin4 (IF OUT) → 믹서 출력
              │              │
Pin7 ────┬────┤ OSC EMI     │
         │    │              │
         L1   └──────────────┘
(T37-6)  │
         │
    ┌────┴────┐
    C2        C3
  (100pF)   (47pF)
    │         │
    └────┬────┘
         │
    ┌────┴────┐
    │  BB109  │ ← 튜닝 전압 (가변저항)
    └─────────┘
         │
        GND
```

### 2. 수신부

#### 2-1. RF 프리앰프
**부품**:
- FET: 2SK125 또는 J310
- 입력 BPF: T37-6 코어 2개, 동조 콘덴서

```
안테나 → [BPF 14MHz] → Gate(2SK125) → Drain → [RFC] → +12V
                              │                    │
                             R(100Ω)              C(0.1μF)
                              │                    │
                             GND              출력 → 믹서
```

#### 2-2. 믹서 (1st Mixer)
**부품**:
- IC: NE602AN
- 입력: RF 프리앰프 출력
- LO: VFO (5MHz)
- 출력: 9MHz IF

#### 2-3. IF 필터 (크리스탈 래더 필터)
**부품**:
- 9MHz 크리스탈: 4~6개 (HC-49/S)
- 결합 콘덴서: 39pF ~ 68pF
- 대역폭: ~400Hz (CW용)

```
IN ──C──┬──[XTAL]──┬──C──┬──[XTAL]──┬──C──┬──[XTAL]──┬──C── OUT
        │           │     │           │     │           │
       39pF        39pF  39pF        39pF  39pF        39pF
        │           │     │           │     │           │
       GND         GND   GND         GND   GND         GND
```

#### 2-4. 프로덕트 디텍터 + BFO
**부품**:
- IC: NE602AN (2번째)
- BFO: 9MHz 크리스탈 발진 (IF 주파수 + 700Hz 오프셋)

#### 2-5. AF 앰프
**부품**:
- IC: LM386N-1
- 게인: 200 (46dB)
- 출력: 스피커 또는 헤드폰

```
프로덕트 디텍터 출력
        │
       C1 (10μF)
        │
Pin3 ───┤ LM386  ├─── Pin5 → C(220μF) → 스피커(8Ω)
        │         │
Pin2 ──GND       Pin6 ── +9V
        │         │
       Pin1──C(10μF)──Pin8  (게인 설정)
```

---

### 3. 송신부

#### 3-1. 키잉 회로
**부품**:
- IC: NE555 또는 트랜지스터 스위칭
- 사이드톤 발진: 700Hz
- 키 입력: 스트레이트 키 또는 패들

#### 3-2. 송신 믹서/버퍼
**부품**:
- VFO 신호를 버퍼링
- 트랜지스터: 2N2222A

#### 3-3. 드라이버 단
**부품**:
- TR: 2SC1971 또는 2SC2078
- 출력: 약 1~2W
- 바이어스: Class C

```
VFO 버퍼 출력
      │
     C (100pF)
      │
  Base(2SC1971)
      │
     R(4.7kΩ) → GND
      
Collector ── [RFC] ── +12V
      │
     C (100pF)
      │
   출력 → PA 입력
```

#### 3-4. 파이널 PA (진공관 6146B)

**이것이 핵심 단입니다!**

**부품**:
- 진공관: 6146B × 1개 (또는 6146W)
- 소켓: 7핀 미니어처 소켓
- 플레이트 전압: 600~750V DC
- 스크린 전압: 200~250V DC
- 그리드 바이어스: -45V ~ -60V
- 출력 탱크: 파이 네트워크

```
                        +HV (600~750V)
                         │
                        RFC (2.5mH)
                         │
                    ┌────┴────┐
                    │  PLATE  │
                    │         │
드라이버 출력       │  6146B  │
    │               │         │
   C(100pF)        │  SCREEN ├──── +250V (R 47kΩ 경유)
    │               │         │
   ├───────────────┤  GRID   │
   │               │         │
  RFC              │ CATHODE ├──── R(10Ω) → GND (전류 측정)
   │               └─────────┘
  -Vbias(-50V)

PLATE 출력:
    │
   C1 (튜닝) ─── L (탱크코일) ─── C2 (로딩) ─── [LPF] → 안테나
  (50~500pF)     (T130-6)        (50~500pF)
```

**파이 매칭 네트워크 값 (14MHz, 50Ω 부하)**:
- C1 (튜닝): 30~300pF 가변
- L: T130-6 토로이드, 1.0mm 선 12회 (약 1.5μH)
- C2 (로딩): 50~500pF 가변

---

### 4. T/R 스위치
**부품**:
- 릴레이: 12V DPDT (Omron G2R-2)
- 다이오드: 1N4148 × 4 (수신부 보호)
- 전환 시간: <10ms

---

### 5. 전원부

#### 저압부 (+12V, +9V)
- 트랜스: 220V → 15V, 3A
- 정류: 브릿지 다이오드 + 4700μF
- 레귤레이터: 7812, 7809

#### 고압부 (플레이트 전압)
- 트랜스: 220V → 550V CT (또는 별도 고압 트랜스)
- 정류: 1N5408 × 4 (브릿지)
- 필터: 47μF/1000V × 2 + 초크 (10H/150mA)
- 블리더 저항: 100kΩ/10W

⚠️ **경고: 고압부는 치명적입니다! 반드시 안전 수칙을 준수하세요.**

## 안테나 권장

50W로 DX 교신을 위해:
1. **다이폴 안테나** (가장 기본): 약 10m × 2 = 20m 전체 길이
2. **인버티드 V**: 중앙 높이 10m 이상
3. **3엘리먼트 야기**: 최고 성능, 로테이터 필요

---

## 제작 순서 권장

1. VFO 제작 및 주파수 확인
2. 수신부 (믹서 → IF 필터 → 디텍터 → AF앰프)
3. 수신 테스트
4. 송신 드라이버 단
5. PA 단 (고압 주의!)
6. T/R 스위치 통합
7. 전체 테스트 및 조정

## 예상 성능

- 수신 감도: ~0.5μV (S/N 10dB)
- 송신 출력: 50W
- 주파수 안정도: ±100Hz (워밍업 후)
- 사이드톤: 700Hz
- 소비 전력: 수신 ~5W, 송신 ~150W

이 스펙이면 20m 밴드 컨디션이 좋을 때 한국에서 미국/유럽 교신 충분히 가능합니다.
