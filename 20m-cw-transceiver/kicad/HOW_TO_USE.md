# KiCad 파일 사용 가이드

## 파일 구성

```
kicad/
├── 20m_cw_transceiver.kicad_pro   ← 프로젝트 파일 (KiCad에서 이걸 열면 됨)
├── 20m_cw_transceiver.kicad_sch   ← 스키매틱 (회로도)
├── 20m_cw_transceiver.net         ← 넷리스트 (부품 연결 정보)
└── HOW_TO_USE.md                  ← 이 파일
```

## KiCad에서 여는 방법

1. **KiCad 8** 을 설치합니다 (https://www.kicad.org/download/)
2. KiCad를 열고 `File → Open Project`
3. `20m_cw_transceiver.kicad_pro` 파일을 선택
4. 스키매틱 에디터가 열리면 회로도를 확인/수정할 수 있습니다

## 스키매틱 → PCB 워크플로우

### 방법 1: 스키매틱에서 직접 (권장)
1. 스키매틱 에디터에서 회로도 확인 및 수정
2. `Tools → Annotate Schematic` (부품 번호 자동 부여)
3. `Tools → Assign Footprints` (풋프린트 확인)
4. `Tools → Update PCB from Schematic` (Ctrl+Shift+U)
5. PCB 에디터가 열리면서 모든 부품이 배치됨 (아직 연결 안 됨)
6. 부품을 수동으로 배치
7. 배선 (라우팅) 수행

### 방법 2: 넷리스트 임포트
1. PCB 에디터를 직접 열기
2. `File → Import Netlist`
3. `20m_cw_transceiver.net` 파일 선택
4. 부품들이 PCB에 로드됨
5. 배치 및 배선 수행

## ⚠️ 중요: 넷리스트만으로는 PCB가 자동 완성되지 않습니다!

넷리스트는 **"어떤 부품의 어떤 핀이 어디에 연결되는지"** 만 알려줍니다.

PCB를 완성하려면 반드시:
1. **부품 배치** — 물리적 위치를 정해야 함
2. **배선 (라우팅)** — 구리 트레이스를 그려야 함
3. **DRC (Design Rule Check)** — 규칙 위반 확인

## PCB 설계 시 주의사항 (이 회로 특유)

### RF 구간 (14MHz)
- 그라운드 플레인 필수 (양면 PCB 권장)
- RF 트레이스는 짧게, 50Ω 임피던스 매칭
- VFO 부분은 차폐 (구리 테이프 또는 별도 차폐함)

### 고압 구간 (6146B PA)
- **최소 간격 3mm 이상** (600V 기준)
- 고압 트레이스 폭: 최소 1.5mm
- PA 부분은 별도 보드로 분리 권장
- 방열 고려 (6146B 소켓 주변 공간 확보)

### 일반
- 바이패스 콘덴서(0.1μF)는 각 IC 전원핀 바로 옆에 배치
- 크리스탈 필터 부분은 밀집 배치 (기생 용량 최소화)
- 입출력 분리 (수신부와 송신부 물리적 분리)

## 추가 필요한 부품 (스키매틱에 미포함)

이 스키매틱은 핵심 신호 경로만 포함합니다. 실제 제작 시 추가 필요:

- 각 IC 전원핀 바이패스 콘덴서: 0.1μF × 4개
- 전원부 (7812, 7809 레귤레이터 + 필터 콘덴서)
- 고압 전원부 (정류 다이오드 + 필터)
- T/R 릴레이 (DPDT 12V)
- 사이드톤 발진기 (NE555)
- 6146B 히터 전원 (6.3V AC)
- 스크린 그리드 전원 (250V + 드롭 저항)
- 입력 BPF (14MHz 대역통과 필터)

## 자동 라우터 사용

KiCad에서 Freerouting을 사용할 수 있지만:
- **RF 회로에는 수동 배선을 강력 권장합니다**
- 자동 라우터는 임피던스 매칭을 고려하지 않습니다
- 고압 간격 규칙도 자동으로 적용되지 않습니다

## 권장 PCB 사양

- 기판: FR-4, 1.6mm 두께
- 구리: 35μm (1oz) — PA 부분은 70μm(2oz) 권장
- 레이어: 최소 2층 (Top + Bottom GND plane)
- 최소 트레이스: 0.25mm (일반), 1.5mm (전력/고압)
- 최소 간격: 0.25mm (일반), 3mm (고압)
